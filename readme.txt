== Description ==

WordPress PHPList Dual Registration allows you to add a checkbox to your WordPress registration page which allows people to also sign up to a chosen newsletter with PHPList. This is a great tool in growing your PHPList user list.

Related Links:

* <a href="http://www.seanbluestone.com/wordpress-phplist-dual-registration-plugin">WordPress PHPList Dual Registration Homepage</a>

== Installation ==

1. Upload the wordpress-phplist-dual-registration folder to your '/wp-content/plugins/' directory.
2. Activate the plugin via the Plugins menu in WordPress.
3. Navigate to Settings -> PHPList Dual Reg. Enter your PHPList URL and other settings.

That's it! Head to your registration page to test it and see it in action.
